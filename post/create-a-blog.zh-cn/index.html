本文是介绍如何使用hugo来搭建自己的属于自己的个人博客站点，最终托管部署至github使用子域名进行访问.

选用Hugo是因其简单而强大的功能，使用其简单的命令和大量丰富的主题可以帮助我们快速构建站点。

搭建环境

1.安装Golang

Hugo基于Go语言，所以你要安装Golang的环境
https://gohugo.io/

2.下载Hugo

安装hugo最通用的方法是到Github下载编译好的二进制文件.
https://github.com/gohugoio/hugo/releases

本人是Mac机型同时有些主题是需要扩展版本才支持所有我选择了 hugo_extended_0.102.3_macOS-universal.tar.gz

3.安装Hugo

Mac系统:
1)下载二进制文件后解压，然后配置PATH环境变量路径即可使用.
2)也可以使用Homebrew进行安装.

Linux系统:
将二进制文件拷贝到/usr/local/bin/文件夹下，这样就可以在命令行中直接运行hugo.

Windows系统:
将二进制文件的路径添加到PATH环境变量中.

如何使用

1.生成站点

1

hugo new site mysite

2.创建文章

1

hugo new post/first.md

3.下载主题

官网主题地址: https://themes.gohugo.io/
把主题通过git克隆或直接下载到本地,放到…/mysite/themes/目录下.

1
2

cd mysite
git clone https://github.com/xianmin/hugo-theme-jane.git --depth=1 themes/jane

4.编译预览

执行命令后，打开网址 http://localhost:1313/ 即可查看本地生成的静态网站

1

hugo server -t jane -D

部署到Github

1.创建仓库

在Github上面创建一个仓库，名字为yourname.github.io格式.(其中yourname是你github的用户名称)

2.Push代码

在你的站点目录下执行，执行完后本地的mysite文件夹下会多出一个public文件夹.

1

hugo --theme=jane --baseUrl='你的远端地址'

注意: 这里没有server，因为我们要部署到远端；theme和上面描述一致；baseUrl=https://yourname.github.io/

进入public文件夹，执行

1

// 进入public文件夹
cd public
// 初始化Git
git init
// 设置远程仓库关联
git remote add origin https://github.com/yourname/yourname.github.io.git
// 设置远程URl,其中token是在github中生成出来的
git remote set-url origin https://token@github.com/yourname/yourname.github.io.git

// 添加代码

// 提交代码
git commit -m "first commit"
// 切换分支
git branch -M main
// Push代码
git push -u origin main

注意：
关联github远程仓库时建议使用 git remote set-url origin https://token@github.com/yourname/yourname.github.io.git,其中token可以在Settings/Developer settings/中生成.

当出现fatal: refusing to merge unrelated histories时使用
git config --global http.sslVerify "false"解决
